# Alura-Play

Passos para rodar o json server. 
* no VSCode - Iniciar o NPM com o comando `npm init`
* no PowerShell - Configurar o PowerShell como adminstrador para receber todas as politicas `Set-ExecutionPolicy RemoteSigned`
* no VSCode - Iniciar o json-server com o comando `npx json-server --watch db.json`
